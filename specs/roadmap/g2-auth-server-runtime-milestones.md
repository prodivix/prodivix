- 只在纵切同时具备 canonical contract、正负向测试、目标矩阵、产品入口（适用时）与相应本地/远端 Gate 后标记 Implemented。
- workflow 存在但尚未运行成功时写 `Configured / Evidence pending`，不得写 `Passed`。
- 详细实现过程和停止条件留在 implementation/ADR；本文件只保存 milestone 判断。
