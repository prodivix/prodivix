# `@prodivix/plugin-antd`

Bundled official Ant Design plugin for Prodivix Blueprint and React export.

The package owns the Ant Design support matrix, six static contribution
resources, deterministic package artifact, build-attested React Host Module,
Palette previews, Canvas provider policy, and icon provider. It does not expose
network loaders or mutate Workspace/PIR state directly.

Run `pnpm generate` after changing the support model. `pnpm check:generated`
fails when resource integrity, catalog metadata, runtime types, creation recipes,
Host implementation declarations, package versions, or generated files drift.
