# GEN-2002 目标框架不支持该能力

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | GEN       |
| 范围     | 代码生成  |
| 严重程度 | `warning` |
| 阶段     | `adapter` |
| 可重试   | 否        |

## 含义

GEN-2002 表示 目标框架不支持该能力。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

PIR 使用了目标框架 adapter 不支持的事件、slot、动画或数据能力

## 建议操作

调整设计或选择支持该能力的导出目标

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
