# PIR-2007 跨结构节点引用不存在

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | PIR      |
| 范围     | PIR 文档 |
| 严重程度 | `error`  |
| 阶段     | `graph`  |
| 可重试   | 否       |

## 含义

PIR-2007 表示 跨结构节点引用不存在。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

PIR 领域字段或 extension protocol 中的节点引用无法解析到 `nodesById`

## 建议操作

重新选择目标节点，或恢复被删除的节点

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
