# ANI-2001 Binding 目标节点不存在

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | ANI       |
| 范围     | 动画      |
| 严重程度 | `error`   |
| 阶段     | `binding` |
| 可重试   | 否        |

## 含义

ANI-2001 表示 Binding 目标节点不存在。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

standalone Animation document 的 target `{ documentId, nodeId }` 无法解析到 PIR document/node

## 建议操作

重新选择动画目标节点或恢复缺失节点

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
