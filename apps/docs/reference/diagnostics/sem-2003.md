# SEM-2003 语义引用解析结果不唯一

## 快速信息

| 名称     | 说明         |
| -------- | ------------ |
| 前缀     | SEM          |
| 范围     | 跨领域语义   |
| 严重程度 | `warning`    |
| 阶段     | `resolution` |
| 可重试   | 否           |

## 含义

SEM-2003 表示 语义引用解析结果不唯一。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

同一优先级内存在多个满足 name、kind、type 与 capability 的候选 symbol

## 建议操作

使用稳定 symbol identity 或收窄作用域与类型约束

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
