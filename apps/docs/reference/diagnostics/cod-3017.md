# COD-3017 Code artifact 已失去 owner binding

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `warning` |
| 阶段     | `binding` |
| 可重试   | 否        |

## 含义

COD-3017 表示 Code artifact 已失去 owner binding。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

由 CodeSlot 托管的 artifact 已不再具有 active binding，例如 slot owner、binding 或对应外部库配置被删除

## 建议操作

将 artifact 重新绑定到兼容 CodeSlot、转为普通 workspace module，或确认不再需要后删除

- 当前项目或工作区 ID

- 可复现时的最小文档或配置
