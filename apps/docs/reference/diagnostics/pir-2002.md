# PIR-2002 节点 key 与节点 ID 不一致

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | PIR      |
| 范围     | PIR 文档 |
| 严重程度 | `error`  |
| 阶段     | `graph`  |
| 可重试   | 否       |

## 含义

PIR-2002 表示 节点 key 与节点 ID 不一致。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

`nodesById` 的 key 与节点内部 `id` 不一致

## 建议操作

重新导入或修复该 PIR 文档
