# UX-9003 UX 检测结果已过期

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `info`    |
| 阶段     | `checker` |
| 可重试   | 是        |

## 含义

UX-9003 表示 UX 检测结果已过期。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

PIR、主题、viewport、资源或运行时状态已变化，现有 UX 诊断不再对应当前预览

## 建议操作

重新运行 UX 检查

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
