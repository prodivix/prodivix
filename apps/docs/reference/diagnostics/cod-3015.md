# COD-3015 代码片段修改了只读上下文

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `error`   |
| 阶段     | `binding` |
| 可重试   | 否        |

## 含义

COD-3015 表示 代码片段修改了只读上下文。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

代码片段尝试写入只读 props、route param、computed input、graph input、PIR snapshot 或受控注入上下文

## 建议操作

改为返回新值、发出允许的 command，或写入当前宿主声明为可变的状态位置

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
