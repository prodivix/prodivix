# COD-3001 代码片段绑定目标不存在

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `error`   |
| 阶段     | `binding` |
| 可重试   | 否        |

## 含义

COD-3001 表示 代码片段绑定目标不存在。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

代码片段 owner 指向的 PIR 节点、Inspector 字段、NodeGraph 节点、Animation track 或其他宿主对象已不存在

## 建议操作

重新选择代码挂载目标，或删除失效代码片段

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
