# SEM-2002 语义引用目标在当前作用域不可见

## 快速信息

| 名称     | 说明         |
| -------- | ------------ |
| 前缀     | SEM          |
| 范围     | 跨领域语义   |
| 严重程度 | `warning`    |
| 阶段     | `resolution` |
| 可重试   | 否           |

## 含义

SEM-2002 表示 语义引用目标在当前作用域不可见。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

symbol 存在，但不在 reference 的 lexical/import scope 中可见

## 建议操作

通过显式参数、binding、import 或合法 owner scope 引入目标

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
