# SEM-2005 语义索引快照已过期

## 快速信息

| 名称     | 说明         |
| -------- | ------------ |
| 前缀     | SEM          |
| 范围     | 跨领域语义   |
| 严重程度 | `warning`    |
| 阶段     | `resolution` |
| 可重试   | 否           |

## 含义

SEM-2005 表示 语义索引快照已过期。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

query 的 expected SemanticSnapshotIdentity 与当前 index identity 不一致

## 建议操作

使用最新 Workspace snapshot 重建查询或计划

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
