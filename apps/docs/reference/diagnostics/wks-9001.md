# WKS-9001 Workspace 未知异常

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | WKS        |
| 范围     | 工作区     |
| 严重程度 | `error`    |
| 阶段     | `snapshot` |
| 可重试   | 是         |

## 含义

WKS-9001 表示 Workspace 未知异常。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Workspace 加载、保存、同步或 replica 恢复中出现未分类异常

## 建议操作

重试操作；若复现，携带错误码和项目 ID 上报

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
