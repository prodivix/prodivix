# WKS-4003 Document revision 冲突

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | WKS       |
| 范围     | 工作区    |
| 严重程度 | `warning` |
| 阶段     | `sync`    |
| 可重试   | 是        |

## 含义

WKS-4003 表示 Document revision 冲突。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

客户端提交的 `contentRev` 或 `metaRev` baseline 落后于服务端，或新增 Document 的 absence precondition 不成立

## 建议操作

查看冲突详情，选择保留本地或远端改动

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
