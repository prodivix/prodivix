# PIR-9001 PIR 未知异常

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | PIR       |
| 范围     | PIR 文档  |
| 严重程度 | `error`   |
| 阶段     | `runtime` |
| 可重试   | 是        |

## 含义

PIR-9001 表示 PIR 未知异常。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

PIR 校验、解析、materialize 或运行时出现未分类异常

## 建议操作

重试操作；若复现，携带错误码和项目上下文上报

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
