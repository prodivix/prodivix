# GEN-1001 Canonical IR 构建失败

## 快速信息

| 名称     | 说明           |
| -------- | -------------- |
| 前缀     | GEN            |
| 范围     | 代码生成       |
| 严重程度 | `error`        |
| 阶段     | `canonical-ir` |
| 可重试   | 否             |

## 含义

GEN-1001 表示 Canonical IR 构建失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

PIR 无法转换为目标无关的 Canonical IR

## 建议操作

先修复 PIR 诊断，再重新导出
