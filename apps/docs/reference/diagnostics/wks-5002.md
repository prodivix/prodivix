# WKS-5002 Patch 应用失败

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | WKS      |
| 范围     | 工作区   |
| 严重程度 | `error`  |
| 阶段     | `commit` |
| 可重试   | 否       |

## 含义

WKS-5002 表示 Patch 应用失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

command envelope 中的 patch path 无法应用到当前 workspace 文档

## 建议操作

刷新工作区并重新执行操作

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
