# COD-2013 引用了当前作用域不可见的符号

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `warning` |
| 阶段     | `symbol`  |
| 可重试   | 否        |

## 含义

COD-2013 表示 引用了当前作用域不可见的符号。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

代码引用的符号存在于项目中，但不在当前 WorkspaceScope、节点上下文、Collection item 上下文、NodeGraph 或 Animation binding 中可见

## 建议操作

将代码移动到正确作用域，或通过参数、props、data scope、graph input 显式传入

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
