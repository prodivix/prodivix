# PLG-2020 Contribution 跨点引用无效

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `semantic` |
| 可重试   | 否         |

## 含义

PLG-2020 表示 Contribution 跨点引用无效。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

External Library、Palette、Blueprint Template、Render Policy、Codegen Policy 或 Icon Provider 引用了未声明的 library、Palette item、runtime type、package、export 或不兼容的 host implementation identity

## 建议操作

让同一插件批次中的扩展点引用精确匹配的 library、Palette item、runtime type、package coordinate、export 和 implementation id

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
