# PIR-3001 ValueRef 路径无法解析

## 快速信息

| 名称     | 说明        |
| -------- | ----------- |
| 前缀     | PIR         |
| 范围     | PIR 文档    |
| 严重程度 | `warning`   |
| 阶段     | `value-ref` |
| 可重试   | 是          |

## 含义

PIR-3001 表示 ValueRef 路径无法解析。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

`$param`、`$state`、`$data` 或 `$item` 指向不存在的路径

## 建议操作

检查绑定的数据源、参数名或列表作用域
