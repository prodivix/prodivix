# PIR-1001 禁止保存树形 UI 根节点

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | PIR      |
| 范围     | PIR 文档 |
| 严重程度 | `error`  |
| 阶段     | `schema` |
| 可重试   | 否       |

## 含义

PIR-1001 表示 禁止保存树形 UI 根节点。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

保存态 PIR 文档中出现 `ui.root`

## 建议操作

在支持当前 `ui.graph` 契约的编辑器中修正并保存文档
