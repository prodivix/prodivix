# EDT-4001 画布预览降级

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | EDT       |
| 范围     | 编辑器    |
| 严重程度 | `warning` |
| 阶段     | `canvas`  |
| 可重试   | 是        |

## 含义

EDT-4001 表示 画布预览降级。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

画布无法完整渲染当前页面，只能显示占位、错误边界或降级组件

## 建议操作

查看诊断详情，优先修复 PIR、外部库或数据绑定错误

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
