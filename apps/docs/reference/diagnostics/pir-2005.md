# PIR-2005 节点存在多个结构父级

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | PIR      |
| 范围     | PIR 文档 |
| 严重程度 | `error`  |
| 阶段     | `graph`  |
| 可重试   | 否       |

## 含义

PIR-2005 表示 节点存在多个结构父级。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

同一节点同时出现在多个 `childIdsById` 或 `regionsById` 位置

## 建议操作

撤销最近一次移动或执行 graph repair

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
