# EDT-1001 当前选中节点不存在

## 快速信息

| 名称     | 说明        |
| -------- | ----------- |
| 前缀     | EDT         |
| 范围     | 编辑器      |
| 严重程度 | `warning`   |
| 阶段     | `selection` |
| 可重试   | 是          |

## 含义

EDT-1001 表示 当前选中节点不存在。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

`selectedNodeId` 无法在当前 PIR graph 中找到

## 建议操作

重新在画布或组件树中选择节点

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
