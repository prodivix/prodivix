# PIR-3010 列表渲染配置非法

## 快速信息

| 名称     | 说明        |
| -------- | ----------- |
| 前缀     | PIR         |
| 范围     | PIR 文档    |
| 严重程度 | `warning`   |
| 阶段     | `value-ref` |
| 可重试   | 否          |

## 含义

PIR-3010 表示 列表渲染配置非法。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

列表数据源、item/index 作用域、stable key 或状态内容不满足当前 PIR 列表渲染契约

## 建议操作

检查列表数据源、item/index 绑定、stable key 和状态内容
