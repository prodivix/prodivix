# PIR-2011 组件组合规则不满足

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | PIR      |
| 范围     | PIR 文档 |
| 严重程度 | `error`  |
| 阶段     | `graph`  |
| 可重试   | 否       |

## 含义

PIR-2011 表示 组件组合规则不满足。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

插入、拖拽、复制、移动或删除导致插件声明的 parent/slot/sequence 组合约束不成立

## 建议操作

保留 compound component 所需的 parent 与 primitive 顺序，或把节点移动到允许的位置

- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
