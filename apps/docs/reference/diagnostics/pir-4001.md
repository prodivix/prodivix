# PIR-4001 Materialize 失败

## 快速信息

| 名称     | 说明          |
| -------- | ------------- |
| 前缀     | PIR           |
| 范围     | PIR 文档      |
| 严重程度 | `error`       |
| 阶段     | `materialize` |
| 可重试   | 否            |

## 含义

PIR-4001 表示 Materialize 失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

`materializeUiTree` 遇到缺失节点、环或重复父级，无法生成完整临时树

## 建议操作

先修复 PIR graph 诊断，再重新预览或导出
