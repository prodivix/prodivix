# PLG-4070 Official component runtime 不可用

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `registry` |
| 可重试   | 是         |

## 含义

PLG-4070 表示 Official component runtime 不可用。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

PIR node 引用 bundled official package 声明的 runtime type，但当前 owner generation 未发布对应 Renderer projection

## 建议操作

启用或重新安装拥有该 runtime type 的 official component package
