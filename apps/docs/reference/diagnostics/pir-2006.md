# PIR-2006 存在未受控孤儿节点

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | PIR       |
| 范围     | PIR 文档  |
| 严重程度 | `warning` |
| 阶段     | `graph`   |
| 可重试   | 否        |

## 含义

PIR-2006 表示 存在未受控孤儿节点。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

`nodesById` 中存在无法从 `rootId` 到达的节点，且未标记为受控扩展

## 建议操作

删除无用节点或把节点重新接入页面
