# PLG-4072 Official component runtime 已不支持

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `registry` |
| 可重试   | 否         |

## 含义

PLG-4072 表示 Official component runtime 已不支持。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

PIR node 使用 official package 明确列入 unsupportedRuntimeTypes 的历史 runtime type

## 建议操作

用当前 official Palette/template 支持的 component 结构替换旧节点
