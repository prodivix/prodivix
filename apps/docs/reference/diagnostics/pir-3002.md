# PIR-3002 数据作用域配置非法

## 快速信息

| 名称     | 说明        |
| -------- | ----------- |
| 前缀     | PIR         |
| 范围     | PIR 文档    |
| 严重程度 | `warning`   |
| 阶段     | `value-ref` |
| 可重试   | 否          |

## 含义

PIR-3002 表示 数据作用域配置非法。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

`data.source`、`data.pick` 或 `data.extend` 不满足 PIR 数据作用域契约

## 建议操作

检查数据源、pick 路径和扩展字段
