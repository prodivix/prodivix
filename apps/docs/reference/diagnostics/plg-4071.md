# PLG-4071 Bundled official library 不存在

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `registry` |
| 可重试   | 否         |

## 含义

PLG-4071 表示 Bundled official library 不存在。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Workspace component library 配置引用 bundled catalog 中不存在的 library id

## 建议操作

删除未知 library id，或安装 Host 明确支持的 bundled official package
