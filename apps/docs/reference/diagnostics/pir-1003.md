# PIR-1003 节点字段非法

## 含义

PIR-1003 表示 节点字段非法。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

`nodesById` 中的节点缺少合法 `id` 或 `type`

重新导入或修复该 PIR 文档
