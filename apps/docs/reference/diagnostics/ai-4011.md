# AI-4011 流式响应结构无法解析

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | AI         |
| 范围     | AI 助手    |
| 严重程度 | `error`    |
| 阶段     | `response` |
| 可重试   | 是         |

## 含义

AI-4011 表示 流式响应结构无法解析。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Provider 流式响应完成，但累计文本无法解析为期望的结构化输出

## 建议操作

重试请求，或在调试详情中查看模型原始返回
