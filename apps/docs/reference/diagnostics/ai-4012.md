# AI-4012 流式响应不可读

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | AI         |
| 范围     | AI 助手    |
| 严重程度 | `warning`  |
| 阶段     | `response` |
| 可重试   | 是         |

## 含义

AI-4012 表示 流式响应不可读。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Provider 声称支持 streaming，但 fetcher 没有返回可读 `ReadableStream`

## 建议操作

重试请求；如果持续复现，切换到非流式 Provider 或检查运行环境
